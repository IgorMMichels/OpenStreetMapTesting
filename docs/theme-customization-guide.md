# Theme Customization Guide

This guide describes how to customize themes for the Vue map application, including color tokens, typography, and runtime theme switching.

- Color tokens:
  - --bg-color: map background
  - --road-color: road network color
  - --text-color: text color
- How to switch themes at runtime:
  - Apply a different CSS class to the map container (e.g., map-dark-theme, map-light-theme).
- Practical example:
  - Update MapLightTheme.vue to apply appropriate tokens when the map is rendered.

For full implementation details, refer to IMPLEMENTATION_PLAN.md.
